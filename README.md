# Portfolio - Base perso epuree

Le projet a ete simplifie pour garder uniquement une base personnelle, propre et facile a faire evoluer.

## Arborescence

- `index.html` : page unique du portfolio
- `styles/index.css` : style principal (palette rose/orange conservee)
- `assets/images/profile.jpg` : photo de profil
- `scripts/` : reserve pour le JavaScript futur

## Direction choisie

- Suppression des pages "scolaires" et galeries.
- Refonte HTML/CSS sur une base one-page plus mature.
- Structure volontairement legere pour iterer vite ensuite (JS puis React si souhaite).
